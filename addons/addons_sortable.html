拖放排序/Sortable

创建可以通过拖拽方式排序的嵌套列表。

它可以让你通过拖拽来对项目进行排序和降级，这种方式非常好，如果你想对不同的类型或菜单方面进行管理，请看下面的示例。

用法

一个可排序的列表包含列表本身及其项目，一个手柄和一个开关。

类/Data属性 概述
.uk-sortable 这个类添加到 <ul> 元素来定义插件。
data-uk-sortable 若要启用必要的JavaScript，只需添加该Data属性。
.uk-sortable-item 这个类添加到列表项里面的每个 <div> 元素。
.uk-sortable-handle 添加这个类来启用手柄和拖拽排序。
data-sortable-action="toggle" 这个Data属性允许你隐藏或显示嵌套的列表项。

示例

  • Item 2
  • Item 3

Code.

<ul class="uk-sortable" data-uk-sortable>
    <li>
        <div class="uk-sortable-item">
            <div class="uk-sortable-handle"></div>
            <div data-sortable-action="toggle"></div>
            ...
        </div>
    </li>
</ul>

排序多个列表

为了让项目能够从一个列表进行排序到另一个，你可以将它们通过添加 data-uk-sortable="{group:'GROUP-NAME'}" 属性到每个列表。默认情况下，没有任何组名的列表会自动在同一个组中。

禁用嵌套

要禁用列表项嵌套，只需添加 data-uk-sortable="{maxDepth:1}" 属性。你也可以使用此Data属性来确定在什么样的深度嵌套是可用的。

  • Item 1
  • Item 2
  • Item 3

Code.

<ul class="uk-sortable" data-uk-sortable="{maxDepth:1}">...</ul>

Javascript 选项

这是关于如何通过属性设置选项的示例：

data-uk-sortable="{maxDepth:0, group:'widgets'}"
选项 可选值 默认值 概述
group 整数 (0..6) 0 列表组
maxDepth 整数 10 最大嵌套级别

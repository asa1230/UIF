Markdown 编辑器/Markdown area

创建一个可即时预览的 Markdown 编辑器 支持HTML语法高亮显示。

这个Markdown 编辑器可以让你在编写Markdown和HTML的同时生成一个实时预览。它包括一个工具栏，可以帮助你改变文本的外观和添加链接、图片、引用文字和列表，而无需编写代码。该编辑器为Markdown和HTML提供语法高亮，你甚至可以切换到全屏模式，这样你就可以不受干扰的工作，静心编写你的内容。

用法

若要应用此插件，你首先需要引用相关的 CodeMirror 和 marked，要做到这一点，只需添加相应的脚本到你的项目头部。

<!DOCTYPE html>
<html>
    <head>
        <title></title>
        <link rel="stylesheet" href="uikit.min.css" />
        <script src="jquery.js"></script>
        <script src="uikit.min.js"></script>

        <!-- Codemirror and marked dependencies -->
        <link rel="stylesheet" href="codemirror/lib/codemirror.css">
        <script src="codemirror/lib/codemirror.js"></script>
        <script src="codemirror/mode/markdown/markdown.js"></script>
        <script src="codemirror/addon/mode/overlay.js"></script>
        <script src="codemirror/mode/xml/xml.js"></script>
        <script src="codemirror/mode/gfm/gfm.js"></script>
        <script src="marked.js"></script>

        <!-- Markdown Area JavaScript and CSS -->
        <script src="htmleditor.js"></script>
    </head>
    <body>
    </body>
</html>

然后，只需添加 data-uk-markdownarea 属性到一个 <textarea> 元素。

Javascript 选项

这是关于如何通过属性设置选项的示例：

data-uk-htmleditor="{mode:'split', maxsplitsize:600}"
选项 可选值 默认值 概述
mode 'split', 'tab' 'split' 查看模式
toolbar Array [ "bold", "italic", "strike", "link", "picture", ... ] 出现在工具栏的按钮列表
maxsplitsize integer 1000 最小值。窗口宽度小于该值时，为选项卡模式，否则自动切换到拆分模式。
lblPreview any string 'Preview' 预览模式的标签字符串
lblCodeview any string 'Markdown' 代码模式的标签字符串

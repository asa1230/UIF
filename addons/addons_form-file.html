文件域/Form file

创建一个让用户选择文件的表单控件。

用法

若要应用此插件，只需添加 .uk-form-file 类到一个容器元素，里面添加一个 button 和 <input type="file"> 元素。你也可以使用任何其他元素来代替 button。

示例

Code.

<div class="uk-form-file">
    <button class="uk-button">...</button>
    <input type="file">
</div>

显示文件名

如果你想在一个 <input> 元素中显示选定的文件名称，只需添加 data-uk-form-file 属性到一个容器元素。

示例

Code.

<form class="uk-form" data-uk-form-file>
    <div class="uk-form-file">
        <button class="uk-button">...</button>
        <input type="file">
    </div>
    <input type="text" disabled>
    <button class="uk-button">...</button>
</form>

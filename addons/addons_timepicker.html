时间选择器/Timepicker

创建一个时间选择器它可以用键盘来很容易的设置使用。

用法

要创建一个时间选择器，只需把 data-uk-timepicker 属性添加到 <input> 元素。该时间选择器会自动显示实际的时间，你可以按键盘的向上或向下键来改变每小时或每刻钟。

示例

Code.

<form class="uk-form">
    <input type="text" data-uk-timepicker>
</form>

显示子午线时间

若要显示子午线时间，只需添加 showMeridian 选项： data-uk-timepicker="{showMeridian:true}"。

示例

Code.

<form class="uk-form">
    <input type="text" data-uk-timepicker="{showMeridian:true}">
</form>

Javascript 选项

这是关于如何通过属性设置选项的示例：

data-uk-timepicker="{minuteStep:5, showSeconds:true}"
选项 可选值 默认值 概述
defaultTime string 'current' 默认值的焦点与空的输入框
minuteStep integer 15 分钟时间间隔
secondStep integer 15 秒的时间间隔
showSeconds boolean false 显示秒
showMeridian boolean false 显示子午线

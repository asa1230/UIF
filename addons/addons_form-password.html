表单密码/Form password

创建一个具有可切换显示或隐藏密码的表单。

用法

要创建一个密码输入表单，只需把添加 uk-form-password 类到你的 <div> 元素。若要创建一个可切换显示或隐藏的密码，只需要添加 .uk-form-password-toggle 类及 data-uk-form-password 属性到 <a> 元素。

示例

Show

Code.

<form class="uk-form">
    <div class="uk-form-password">
        <input type="password">
        <a href="" class="uk-form-password-toggle" data-uk-form-password>...</a>
    </div>
</form>

Javascript 选项

这是关于如何通过属性设置选项的示例：

data-uk-form-password="{lblShow:'...', lblHide:'...'}"
选项 可选值 默认值 概述
lblShow any string 'Show' 隐藏标记文本
lblHide any string 'Hide' 显示标记文本

日期选择器/Datepicker

创建一个可切换具有下拉的日期选择器。

用法

要创建一个日期选择器，只需把 data-uk-datepicker 添加到 <input> 元素。您还可以根据自己的需求来自定义日期格式，只需要添加 format 选项到 data-uk-datepicker 属性。

示例

Code.

<form class="uk-form">
    <input type="" data-uk-datepicker="{format:'YYYY.MM.DD'}">
</form>

Javascript 选项

这是关于如何通过属性设置选项的示例：

data-uk-datepicker="{weekstart:0, format:'DD.MM.YYYY'}"
选项 可选值 默认值 概述
weekstart 整数 (0..6) 1 一周的开始
i18n JSON 对象 { months:['January',...], weekdays:['Sun',..,'Sat'] } 语言字符串定义
format DD，MM 和 YY 的任意组合 'DD.MM.YYYY' 日期格式字符串
offsettop 整数 5 输入偏移值

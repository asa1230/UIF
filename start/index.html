开始使用

熟悉 UIF 的基本组织结构。

首先，你需要下载 UIF。在 GitHub 上，你可以找到整个项目的所有源文件。

下载 UIF

文件结构

在ZIP文件中，你可以找到准备在你的项目中使用的所有的CSS、JavaScript和字体文件。UIF 的核心框架几乎没有任何样式，这样做是为了保持它的苗条。因此我们提供两个额外的风格样式，一个渐变样式和一个扁平化样式。每个风格样式提供单独的CSS文件以及压缩后的版本。

文件夹 概述
/css 包含 UIF 所有的CSS文件和压缩后的版本。
/images 包含 UIF 所有的Images文件和压缩后的版本。
/js 包含 UIF 所有的JavaScript文件和压缩后的版本。 
/css
    <!-- UIF的基本样式 -->
    reset.css
    component.css       //组件公共样式，后面组件以单独样式表存在，后面统一压缩到component.css样式表；

   
 

/js
    <!-- JavaScript和压缩后的版本 -->
   jquery-1.6.4 .min.js
   component.js       //组件公共js库，后面组件以单独js库存在，后面统一压缩到component.js文件中

HTML页面代码

首先，确保你有一个可靠的代码编辑器，比如 Sublime Text 2 。你需要在你的HTML5的文档头部添加编译后的UIF CSS和JavaScript文件，最好是压缩后的版本。别忘了添加 jQuery 。就是这样！

示例

<!DOCTYPE html>
<html lang="ch" >
    <head>
        <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
	    <meta name="renderer" content="webkit" />
	    <meta http-equiv="X-UA-Compatible" content="IE=Edge,chrome=1" /> 
        <title></title>
        <link rel="stylesheet" href="../../static/common/css/reset.css" />
        <link rel="stylesheet" href="../../static/common/css/component.css" />
        <script src="../../static/common/js/jquery-1.6.4 .min.js"></script> 
    </head>
    <body>
    </body>
</html>

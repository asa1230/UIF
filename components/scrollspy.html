监听滚动/Scrollspy

页面滚动时触发事件及动画。

用法

该监听滚动组件监听页面滚动，并触发基于滚动位置的事件。比如当你向下滚动页面时，你可以使首次出现在视窗中的一个元素触发一个平滑淡出的动画。

Data属性 概述
data-uk-scrollspy="{cls:'MY-CLASS'}" 只有元素首次出现在视窗时才触发。
data-uk-scrollspy="{cls:'MY-CLASS', repeat: true}" 每次元素出现在视窗中都触发。
data-uk-scrollspy="{cls:'MY-CLASS', delay:600}" 添加以毫秒为单位的动画延迟。

通常情况下，动画组件 的类都与监听滚动一起使用。

Code.

<div data-uk-scrollspy="{cls:'uk-animation-fade', repeat: true}">...</div>

<div data-uk-scrollspy="{cls:'uk-animation-fade', delay:900}">...</div>

Javascript 事件

该监听滚动组件，每次当一个元素出现在视窗中时触发 uk.scrollspy.inview 事件，元素离开视窗时触发 uk.scrollspy.outview 事件。

示例

$('[data-uk-scrollspy]').on({

    'uk.scrollspy.inview', function(){
        console.log("Element is visible.");
    },

    'uk.scrollspy.outview', function(){
        console.log("Element is not visible.");
    }
});

监听滚动导航/Scrollspy Nav

根据你网站滚动条的位置来自动更新所对应的导航标记，只需添加 data-uk-scrollspy-nav 这个data属性到任意导航中。导航项中的链接必须对应到网站中具有相同ID的部分。

Data属性 概述
data-uk-scrollspy-nav 触发监听滚动导航功能所必需的JavaScript。
data-uk-scrollspy-nav="{closest:'MY-SELECTOR'}" 通过遍历查找此DOM树中最接近的节点元素进行匹配。
data-uk-scrollspy-nav="{smoothscroll:true}" 在网站的不同部分之间切换时应用 平滑滚动组件。
data-uk-scrollspy-nav="{cls:'MY-CLASS'}" 默认情况下ScrollspyNav将切换 uk-active 类。使用 cls 选项可以使用自己的类名。

对于监听滚动导航的一个例子, 可以查看 监听滚动导航测试页。

Code.

<ul class="uk-nav uk-nav-side" data-uk-scrollspy-nav="{closest:'li', smoothscroll:true}">
    <li><a href="#MY-ID">...</a></li>
    <li><a href="#MY-ID2">...</a></li>
</ul>

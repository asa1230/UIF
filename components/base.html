基础

提供所有的HTML元素默认样式。

这个组件定义了你的网页的基础视觉效果。通过为每个HTML元素设置默认的颜色、边距、字体大小及其他的属性实现强大的版面效果。本页是一个关于如何使用基础HTML元素和 UIF 如何对它们进行风格化的简要向导。

标题

使用 <h1> 到 <h6> 标签来定义你的标题。

示例

h1 Heading 1

h2 Heading 2

h3 Heading 3

h4 Heading 4

h5 Heading 5
h6 Heading 6

你可以添加 .uk-h1，.uk-h2，.uk-h3，.uk-h4，.uk-h5 或 .uk-h6 类来改变你的标题的大小，比如让 h1 看起来像一个 h3。

段落

通过自定义设置变量，可对全局的字体大小、行间距、边距等常规元素进行调整。段落和其他块元素也继承这些变量值。

文本层次语义

下面的列表为你提供了最常用的文本层次语义，以及如何利用它们的简短概述。

标签 概述
<a> 将文本转换为超文本使用 a 标签。
<em> 强调文本使用 em 标签。
<strong> 强调任何额外的更重要文本使用 strong 标签。
<code> 定义内联代码片段使用 code 标签。
<del> 标记文档中已被修改或删除的文本使用 del 标签。
<ins> 标记文档中插入的文本使用 ins 标签。
<mark> 高亮显示文本使用 mark 标签，它没有任何语义含义。
<q> 定义引入的语录在 q 标签 里面使用 q 标签。
<abbr> 定义一个标题的缩写使用 abbr 标签。
<dfn> 定义一个项目术语名称使用 dfn 标签。
<small> 缩小显示不重要的文本使用 small 标签。

水平线

使用 <hr> 标签创建一条水平线。

示例

引用文字

示例

Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam.

代码块

对于多行代码，使用 <pre> 标签定义含有固定格式的文本。它会建立一个保留空格、制表符及换行符的文本块。内部嵌套 <code> 标签来定义代码块。
重要 为保证正确地渲染，必须确保避开代码中的任何尖括号。

示例

<pre>
    <code>...</code>
</pre>

注意 你也可以添加 通用组件 中介绍到的 .uk-scrollable-text 类, 这将设置最大高度为300px，并提供Y轴滚动条。

描述列表

创建一个描述列表使用 <dl> 标签。使用 <dt> 定义项目，使用 <dd> 定义描述内容。

示例

Description lists
A description list defines terms and their corresponding descriptions.
This is a term
This is a description.
This is a term
This is a description.

Code.

<dl>
    <dt>...</dt>
    <dd>...</dd>
</dl>

表格

通过 <table> 标签创建一个表格。使用 <tr>， <td> 和 <th> 标签建立表格行，表格单元格和表格的标题。<thead>， <tbody> 和 <tfoot> 标签定义表格的部分。你还可以通过添加 <caption> 标签来创建标题。

示例

Table caption
Table Heading Table Heading Table Heading
Table Footer Table Footer Table Footer
Table Data Table Data Table Data
Table Data Table Data Table Data
Table Data Table Data Table Data

Code.

<table>
    <caption>...</caption>
    <thead>
        <tr>
            <th>...</th>
        </tr>
    </thead>
    <tfoot>
        <tr>
            <td>...</td>
        </tr>
    </tfoot>
    <tbody>
        <tr>
            <td>...</td>
        </tr>
    </tbody>
</table>

响应式图片

UIKit 中的所有图片默认都是响应的。如果布局变窄，图片会调整它们的大小并保持自身的比例。

示例

调整浏览器窗口大小观察图片的响应式行为。

注意 若要避免响应式行为并保持图像的原始尺寸，可以为单独的图片添加 .uk-img-preserve 类。如果你具有多张图片，也可以将这个类添加到父容器中。谷歌地图不需要这样的设置。
